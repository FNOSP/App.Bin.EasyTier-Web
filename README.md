# EasyTier-Web飞牛套件
- EasyTier是一个由 Rust 和 Tokio 驱动的简单、安全、去中心化的异地组网方案。
- EasyTier-Web是为EasyTier提供web管理页面，让用户更直观地管理组网。
### 使用方法：
API主机填飞牛本机ip+11211端口，例如http://192.168.1.3:11211。 初始用户名和密码都是admin，点击登陆后，找到左边侧边栏的设备列表，添加一个新的网络，剩下的操作和其他客户端一样，详情可以看easytier官网介绍。
